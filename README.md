## Setup

### Requirement

You need to have this

- npm
- nodejs

### How to setup locally

- Clone this repository

```bash
git clone git@github.com:satriadhikara/Tucil2_13522125_13522135.git
```

- Install the required dependencies

```bash
npm install
```

- Run

```bash
npm run dev
```
